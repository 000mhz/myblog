アシモフの雑学コレクション

久々に見つけてパラパラとめくって読んでみました。ありとあらゆることの雑学が短い文に収めてある暇つぶしにもってこいの本。アシモフ×星新一×真鍋博という豪華なメンツで構成されてるのもかなり良いです。
どれも面白いんですけど、かつてかなり発達した文明が滅びたってのがあったような気がして読み返してたんですが、見つからず。。その話が普通の滅び方でなかったので記憶に残っているのですが、なぜか見当たらない。。
もう一回読んでみようかな。。

植物

去年の11月あたりに買った観葉植物なんですが1週間もしたら葉がぱさぱさになって枯れ始めていました。しかし、水をやり日のあたるところにおいていると新しい葉や芽が出てきて復活を遂げました。
買ってきた時は手のひらサイズだったのが一回りは大きくなりました。
実は結構植物を育てるのは好きで、200円くらいの鉢を買ってきてでかくなるまで育てるのが楽しみだったりします。でかくなり過ぎて鉢を変えるのは面倒だったりしますが、育てるのが楽しいので苦にならなかったりします。
ちなみにこの植物の名前は不明です。図鑑で調べたりしたんですが載ってなかったです。。

ドラマなど日常

何が楽しいってドラマ面白いです。『悪夢ちゃん』以来ドラマを見るようになって5本くらい見てます。全部録画してるのですが消化するのが意外と大変です。ホームネットワークを作って PC で録画した番組を見れるように出来れば夜中とかに見れるので隙があれば調べて構築したいと思ってます。
肝心のドラマですが、最近のドラマはテンポがすごい速くて一昔なら来週に引っ張ったような内容をさくっと進めていくのは結構見ていてストレスがなくていいです。
あと最近楽しいのは空き時間に API 関連をいじくって自己満サービス試したりしています。去年はあまり本質的に理解していないで API を使っていたのですが、いろいろなサービスの API をいじくりこれはこうできるなど考えてやってみると理解出来面白いです。
画像は最近頻繁に食べてるカリカリ梅です。
