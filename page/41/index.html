ドラマなど日常

何が楽しいってドラマ面白いです。『悪夢ちゃん』以来ドラマを見るようになって5本くらい見てます。全部録画してるのですが消化するのが意外と大変です。ホームネットワークを作って PC で録画した番組を見れるように出来れば夜中とかに見れるので隙があれば調べて構築したいと思ってます。
肝心のドラマですが、最近のドラマはテンポがすごい速くて一昔なら来週に引っ張ったような内容をさくっと進めていくのは結構見ていてストレスがなくていいです。
あと最近楽しいのは空き時間に API 関連をいじくって自己満サービス試したりしています。去年はあまり本質的に理解していないで API を使っていたのですが、いろいろなサービスの API をいじくりこれはこうできるなど考えてやってみると理解出来面白いです。
画像は最近頻繁に食べてるカリカリ梅です。

『ラジ録2』が不調な時は．．．

最近『ラジ録2』の調子が悪い時が多く、途中で途切れるやなんか音が変になるといった症状が現れます。問い合わせて色々やってみたのですがどうも改善される気配がないので困ったなと思っていたのですが、意外なところから解決法がありました。
MP3 データしか保存していなかったのですが flv ファイルも保存しておくようにします。そして保存に失敗したデータを iExtractMP3 にぶち込みます（flv ファイルの方です）。すると m4a のファイルが出来上がり壊れていたファイルが聴けるようになります。音質は若干落ちてるかと思いますが、聴ければ問題ないとおもってるのでこれで代用しています。

坂の上の雲

部屋を掃除していたら父から借りた『坂の上の雲』を見つけ「さすがに読まないとなぁ（2年前くらいに借りた）」と思い、土曜から読み始めました。NHK でドラマ化され放送していたのを途中から見ていたので、借りたとき読むのをためらっていたのですが、借りた以上いい加減読まねばという気持ちが出てきたので読破する予定です。8巻もあるのでなかなか大変ですが、そんなに難しい文体でもないのでいけそうです。
