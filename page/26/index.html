ハートがヒートしてきたぜ

800字を目標にこの日記を続けているのですが、これといった核になるのがないので毎回だらだらと書いてます。今回もだらだらといきます。
新潮社から出ている星新一さんの本はだいたい40冊くらいあるらしいのですが、いまのところ32冊くらい集めました。よく行く古本屋に大量に売られていたのでちょこちょこ買っていたのですが、気がつけばあっという間に集まってました。
和田誠さんや真鍋博さんの装丁も良いのですが、なんといってもショートショートは手軽に読めて面白いです。
星さんは意図的に時事風俗を排除しているのですが、最終的な落としどころは現代の寓話らしいです。という文章を発見してなるほどと感心しました。単純にひねりの利いたショートショートではなく、寓話にしたいという思いは創作の力になってたのではないかと思いました。
う〜ん素晴らしいモチベーションですね。こう言ったことを発見すると結構刺激されてなんか作りたい欲求にかられます。
なんか役立つこと？面白いこと？なんて考えながらやってるとだいたいドツボにはまってモチベーション落ちたりしますよね。こういうのは単純な足し算みたいにあれとこれを合わせてどーんみたいな感じが良かったりするような気がします。勢いが大事だったりね。
というわけでいくつかやってみようと思ってることが出来たので徐々にやっていこうと思います。しかし、僕には技術がないので完成には時間かかるなぁ。こういうとき誰かと組んでさくっとやってもらうのありだと思うのですが、勉強もかねて自分でやって力つけようと思います。
PHP と CSS 、javascript あたりで出来るかな〜週末プログラマみたいなものなんで勉強することがいっぱいだな。こつこつやっていこう。
結論として良質なインプットはアウトプットのきっかけになるのかなと感じてます。最終的に完成させてなんぼのとこもありますけどね。


ナポリタン食べたい。

本買ってます

勉強のために動物本の『デザイニング・ウェブインターフェース』と『プログラマの数学』を買い読んでました。


動物本は難しいと思っていたのですが、読んでみるとなるほどこうやってるからこのサイトは使いやすいのかということがよくわかりました。小難しいカテゴライズがややこしいですが、なんとなく理解して読んでます。ごりごりの Ajax でリッチに作っているのを解説しており、個人的にはあまり Ajax とか javascript をやってないので結構勉強になります。


『プログラマの数学』は高校時代結構避けてきた分野の数学だったので改めて勉強になりました。大学受験の時は文系なのに数学で受けたので結構勉強したのですが、いろいろと抜けてますね。なんというか当時は暗記でどうにか乗り切ったのですが、やっぱりそれだと応用がつらくてプログラミングやってるときも限界がすぐに来るので、改めて勉強してみて良かったです。


あと漫画で『ヤンキー塾へ行く』の2巻が出たので購入しました。室田先輩面白過ぎだな。丸善に買いに行ったのですが、新刊の売り場に置いてなくてひっそりと棚にあったのでもっと盛り上がってほしいですね。ヤンマガ毎週買って読んでるんですが『ヤンキー塾へ行く』の部分スクラップして読み返してます。本当面白いです。


ipod touch の kindle アプリで結構本読んでます。小説とかは読むのに苦痛ではないのですが漫画などはつらいなと感じてきたのでタブレット調べてみました。どうやら Nexus 7 か kindle fire あたりが良さそうです。ipad mini でも良いのですが高いですね。タブレット導入したら本ががっつり電子書籍に乗り換えられるので部屋の本のスペースが空いて良さそうです。なんだかんだで小説とかは読みたいからなぁ。
ここ最近は音楽も iTunes Store とか amazon の music store とか CDじゃなく音源ダウンロードで買うようになってきましたね。レッツデジタル化！


ギョウザ食べたい。

物欲大爆発

あぶく銭みたいなのが入って長年使ってた財布などを買い替えました。正直かなりひどい案件で思い出したくもないのでパーっと使ってしまおうと思い散財してます。
散財と言っても財布はたぶん8年くらい使い続けたのでぼろぼろになりつつあるし、さすがに買い替えたいと思っていたところでした。
ほかにもズボンを一本追加したいと思いグラミチのを購入。普段はユニクロか無印なので高いな〜と思って履いてみたんですが、さすがクライミングのメーカーだけあってゆったり感が素晴らしいですね。高いだけあるわ。素材もツイードなんだけどごま塩でいい感じです。
あと欲しいのはメガネですかね。レンズは交換して使ってたんですが、さすがにフレームも買い替えたいなと思ってます。財布と同じで8年くらい使い続けたのでそろそろ交換時かなと思います。
こうやって書いてみると、大体今使ってる身の回りの物は大学時代に買って使ってる物が多いですね。あのときはバイトしてようやく金銭的に余裕が出てきた時期だったんでしょうね。サラリーマン時代は金は入ってくるけど、使う暇が本当になくてお金全く使わなかったです。
大きな買い物と言えば Mac くらいなものでほんとひっそり暮らしてます。
もう少し金ためてアメリカ行きたいなぁ。メジャーとNBA 見たいもんな〜
『釣れんボーイ』の主人公が金を使ってその反動で仕事をするって描いてあったけど、さすがにそういう感じではないですが、目標もってモチベーション高めて仕事するのは悪くないなと思いますね。
買いたい物はいろいろあるんだけど、衝動的に買い続けると買い物依存症みたいで怖いなと思って制限かけてます。買えば満足するんだろうけど、買わなくても特に不自由しないかもと思うタイプなので買わなくてもいいかと結論づけてます。
といっても制作環境が古いのでもう少し頑張って Mac mini 買って新しいことやってみようかと考えてます。
