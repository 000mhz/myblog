肉の日

過去の日記を移行していたが、とてつもない量があり断念した。無駄だなぁと思いながらも、なんとなくサイトいじるくらいしか思いつかないのでまた作業を再開した。
夏の終りに撮った壁には葉が茂っていたが例年と同じくこの季節では散っている。なにかしているようで何もできなくなる。
時間はあっという間だ。
頓挫していることはいくつかあるが、馬力が出ない。。
野球のシーズン前になんとか一つ終わらせたい。

GitHub Pagesでblogを再開する

初めの会社に務めていた時にMovable Typeではじめたblogは数年前にWordPressへ移転したところで力尽きた。
テーマを作るのが面倒、画像の処理や不要なタグ、プラグインなどあらゆる作業が苦痛である。
そこでGitHub PagesにHexoを使い、Markdownで統一して記事を書くことにした。シンプルに記述し、記録する程度で十分かなと。
blogで書くことはあまりないかもしれないが、仕事がほぼ印刷物なのでどこかでWebのことも触っておきたいなぁと感じたため。まぁいまやフロントエンドのことなんておいそれとできないなぁとは思いつつ、JavaScriptの勉強はしときたいと思っている。InDesignを始めとしたAdobeのソフトはJavaScriptで拡張できることが多く、勉強しておいて損はなさそう。

6月まとめ

6月分を振り返るといってもほとんど外出することもなく溜まっていた週刊誌を読み続けていた。単行本で漫画を買うよりも連載で読んでいるとスピード感はもちろんなんだけど、週刊誌自体が持つ雰囲気がいい。あの連載が終わったとか新しい連載が始まったとか、週刊誌の持つ生態系みたいのを見れるとこかな。
一応備忘録なんでその他のことも少し。

◎映画

『ディストラクションベイビーズ』

映画館で観てきた。
バイオレンスな映画だなぁとビビる。
今の流行りっぽい撮影の仕方ではないような感じだし、バイオレンスなんだけど銃で撃たれるとかそう言った非日常感はなく、素手で殴り続けるところが逆にリアルさを醸し出してた。いきなり物語に投げ込まれて放り出される感じかなぁ。しかし、小松菜奈さんの役どころえぐかったなぁ。悪すぎだわ。。
https://distraction-babies.com/
