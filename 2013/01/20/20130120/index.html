ドラマなど日常

  • Author Dai Suzuki
  • Date Jan 20
  • Categories no_categories

何が楽しいってドラマ面白いです。『悪夢ちゃん』以来ドラマを見るようになって5本くらい見てます。全部録画してるのですが消化するのが意外と大変です。ホームネットワークを作って PC で録画した番組を見れるように出来れば夜中とかに見れるので隙があれば調べて構築したいと思ってます。
肝心のドラマですが、最近のドラマはテンポがすごい速くて一昔なら来週に引っ張ったような内容をさくっと進めていくのは結構見ていてストレスがなくていいです。
あと最近楽しいのは空き時間に API 関連をいじくって自己満サービス試したりしています。去年はあまり本質的に理解していないで API を使っていたのですが、いろいろなサービスの API をいじくりこれはこうできるなど考えてやってみると理解出来面白いです。
画像は最近頻繁に食べてるカリカリ梅です。
